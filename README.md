# The Archivist (Shortened)

The Archivist is a web-based psychological thriller game. You are a technician transferring corrupted files from an old data server to a new one. But beware. An AI anomaly lurks between. 

In this version, only one file is required to be transferred to reach the endings. It can be played [here](https://nitya-desaraju.github.io/the_archivist_endings/). The full version can be played [here](https://nitya-desaraju.github.io/the_archivist_improved/).
